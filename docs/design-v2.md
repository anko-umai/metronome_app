# メトロノームアプリ V2 設計書

## 1. 概要

V1のメトロノーム機能に加え、以下の新機能を追加する。

1. **使い方ガイド** - アプリの操作方法を解説するコンテンツページ
2. **リズム練習のコツ** - 効果的な練習方法を解説するコンテンツページ
3. **リズムエディター** - 五線譜上に音符・休符を配置し、メトロノームと共に再生確認できる機能

## 4. リズムエディター設計

### 4.1 画面構成

```
+----------------------------------------------+
|  [メトロノーム] [リズムエディター] [ガイド] [コツ]  |
+----------------------------------------------+
|            リズムエディター                     |
+----------------------------------------------+
|  テンポ: 120 BPM  [-]=====[+]                 |
|  拍子: [2] [3] [4*] [5]                       |
+----------------------------------------------+
|                                              |
|  ═════════════════════════════════           |
|  ═════════════════════════════════  (五線譜)  |
|  ═══●═══●═●═══●═════════════════           |
|  ═════════════════════════════════           |
|  ═════════════════════════════════           |
|                                              |
|  残り: 2 拍                                   |
+----------------------------------------------+
|  音符: [2分] [4分] [8分] [16分]                |
|  休符: [2分休] [4分休] [8分休] [16分休]          |
|                                              |
|  [← 削除] [クリア]                             |
+----------------------------------------------+
|  プリセット:                                   |
|  [基本4分] [基本8分] [16分+8分] [休符入り]       |
|  [シンコペーション]                             |
+----------------------------------------------+
|  [▶ 再生]        [メトロノーム ON/OFF]          |
+----------------------------------------------+
```

### 4.2 データモデル

```javascript
// リズムパターン: イベントの配列
// duration: 4分音符 = 1, 8分 = 0.5, 16分 = 0.25, 2分 = 2
pattern = [
  { type: 'note', duration: 0.25 },  // 16分音符
  { type: 'note', duration: 0.25 },  // 16分音符
  { type: 'note', duration: 0.5 },   // 8分音符
  { type: 'rest', duration: 1 },     // 4分休符
  ...
]
// 合計duration = beatsPerMeasure であること
```

### 4.3 五線譜レンダリング (SVG)

- SVG viewBox: 800 x 150
- 5本の水平線（12px間隔）
- 拍子記号表示
- 拍境界の点線ガイド
- 音符: 符頭（楕円）+ 符幹（縦線）+ 符尾（旗 or 連桁）
- 休符: 簡略化したSVGシェイプ
- 同一拍内の連続した旗付き音符は連桁（ビーム）で表示
- 再生中は現在の音符をオレンジ色でハイライト

### 4.4 音声設計

| 音種 | 周波数 | 持続 | 音量 | 用途 |
|------|--------|------|------|------|
| パターン音 | 660 Hz | 80ms | 0.8 | リズムパターンの音 |
| アクセント | 880 Hz | 60ms | 0.6 | メトロノーム1拍目 |
| 通常拍 | 440 Hz | 60ms | 0.4 | メトロノーム2拍目以降 |
