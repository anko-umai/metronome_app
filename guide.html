メトロノーム リズムエディター ガイド 練習のコツ

使い方ガイド

1. メトロノームの基本操作

テンポの設定

テンポは 40〜200 BPM の範囲で設定できます。

  • スライダー を左右に動かして大まかに設定
  • +/- ボタン で1 BPMずつ微調整
BPM（Beats Per Minute）は1分間の拍数です。数値が大きいほど速いテンポになります。

テンポの目安

速度記号BPM雰囲気
Largo40-60非常にゆっくり
Adagio60-76ゆっくり
Andante76-108歩くような速さ
Moderato108-120中くらい
Allegro120-156速い
Presto168-200非常に速い

拍子の選び方

拍子ボタンで 2・3・4・5拍子 を選択できます。

  • 2拍子 - 行進曲、マーチ
  • 3拍子 - ワルツ、メヌエット
  • 4拍子 - ポップス、ロック（最も一般的）
  • 5拍子 - ジャズ、変拍子の練習
1拍目は高い音（880Hz）と赤いインジケーターで強調されます。小節の始まりを把握しやすくなっています。

音符モードの活用

音符モードを切り替えると、1拍あたりの細分化が変わります。

音符1拍の分割活用場面
2分音符2拍に1回ゆったりした拍の確認
4分音符1拍に1回基本テンポの確認
8分音符1拍に2回裏拍の練習
16分音符1拍に4回細かいパッセージの練習
3連符1拍に3回3連符のリズム感の養成

2. リズムエディターの使い方

パターンの作成

  1. テンポと拍子を設定する
  2. 「音符」または「休符」ボタンで音符の種類を選ぶ
  3. ボタンをクリックすると五線譜上にパターンが追加される
  4. 小節が完成するまで音符・休符を追加する
  5. 「再生」ボタンでリズムを確認する

付点音符・付点休符

付点音符は元の音符の1.5倍の長さになります。

付点音符長さよくあるセット
付点2分音符3拍3/4拍子で1小節分
付点4分音符1.5拍付点4分 + 8分 = 2拍
付点8分音符0.75拍付点8分 + 16分 = 1拍

操作ボタン

  • 削除 - 最後に追加した音符・休符を1つ削除
  • クリア - パターンを全て消去
  • プリセット - よく使うリズムパターンを読み込む
  • メトロノーム ON/OFF - 再生時のメトロノーム音を切替

プリセットパターン

名前内容練習用途
基本4分4分音符 x 4テンポの基本確認
基本8分8分音符 x 8裏拍の練習
16分+8分16分x2 + 8分 x 4混合リズムの練習
休符入り4分 + 4分休 + 8分x2 + 4分休休符の感覚を養う
シンコペ8分 + 4分 + 4分 + 4分 + 8分シンコペーションの練習
付点リズム付点4分 + 8分 x 2付点リズムの練習

3. 効果的な使い方のヒント

  • まず遅いテンポ（60〜80 BPM）で正確に弾けるようにする
  • 慣れたら少しずつテンポを上げる（5〜10 BPMずつ）
  • リズムエディターでパターンを確認してから、メトロノームで練習する
  • 苦手なリズムはリズムエディターのプリセットで繰り返し聴く
